Accountability: About This Data Set

This dataset contains information of power plants around the world. Each entry The Database currently contains over 28,500 power plants in 164 countries, representing about 80% of the world's capacity. The cleaned dataset contains country code, full country name, name of power plant, the capacity of power plant, longitude, latitude, 3 types of fuels used, and estimated electricity generation for each power plant

Where did this data come from?

The original dataset was download from Global Power Plant Database. The dataset was then cleaned up using Wrangler and loaded to a Google Fusion Table.

Who has access to it?

The original dataset is to the public. The Google Fusion Table created with the cleaned data is set to open to the public and anyone from web can access to it.
